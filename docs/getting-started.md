# Getting Started

Welcome to the W-AI Platform! This guide will help you set up your local environment.

## Prerequisites
* Git
* Node.js (for running examples and templates)
* OpenAI API Key (for running AI agents locally)

## Quick Start
1. **Clone the repo**:
   ```bash
   git clone https://github.com/AIsulab/ai-mvp.git
   cd ai-mvp
   ```
2. **Explore the Prototype**:
   Navigate to `examples/dashboard-prototype/` and open `index.html` in your browser.
3. **Configure Agents** (Coming Soon):
   Set your `OPENAI_API_KEY` in your environment and explore the `agents/` directory to see how we automate development.
