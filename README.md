# W-AI Dashboard (사장님을 위한 AI 운영 플랫폼)

## 소개
이 프로젝트는 소상공인(사장님)들이 손쉽게 매장을 마케팅하고 관리할 수 있도록 돕는 AI 기반 운영 플랫폼 대시보드(프로토타입)입니다. 기상청 공공데이터를 활용하여 실시간 날씨 기반 맞춤형 마케팅 문구를 생성하고, SNS 콘텐츠 기획 및 리뷰 답변 작성을 자동화하여 업무 효율을 높이는 데 목적이 있습니다.

## 주요 기능
- **날씨 마케팅 자동 생성**: 날씨(비, 맑음, 폭염, 미세먼지 등)에 맞춘 특별 프로모션 문구를 AI가 즉시 생성
- **SNS 콘텐츠 기획**: 계절/상황에 맞는 매장 홍보용 SNS 콘텐츠 자동 제안
- **리뷰 답변 자동화**: 고객 리뷰에 대한 자연스러운 맞춤형 답변 작성 기능
- **실시간 데이터 대시보드**: AI 생성 내역, 실시간 공공 API 연동 상태, 지원금 매칭 현황 등 직관적인 UI/UX 제공

## 사용 방법
1. **다운로드 (저장소 클론)**
   ```bash
   git clone https://github.com/AIsulab/ai-mvp.git
   ```
2. **실행**
   - **로컬 실행**: 다운로드한 폴더 내의 `index.html` 파일을 더블클릭하여 브라우저에서 엽니다.
   - **웹 접속**: Vercel에 연동되어 있다면 제공받은 Vercel 배포 URL로 바로 접속합니다.
