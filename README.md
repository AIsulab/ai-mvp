## GitHub + Vercel 배포 방법

1. 이 폴더를 GitHub 저장소에 업로드
2. Vercel 접속 → Add New Project
3. GitHub 저장소 선택 후 Import
4. Deploy 클릭 (설정 변경 없이 바로 배포 가능)

배포 후 주소 예시: https://your-project.vercel.app
