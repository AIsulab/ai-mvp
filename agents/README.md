# AI Agents & Prompts

This directory contains scripts, prompt templates, and agent configurations designed to interact with models like OpenAI Codex.

## Purpose
The goal is to provide ready-to-use agents that can:
1. Generate UI components based on descriptions.
2. Automate testing of generated code.
3. Scaffold entire MVPs from a set of business requirements.

*(Active development for Phase 2. Contributions welcome!)*
