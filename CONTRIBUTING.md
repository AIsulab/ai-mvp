# 기여 가이드라인 (Contributing to AI-MVP)

AI-MVP(마케팅 자동화 파이프라인) 프로젝트에 오신 것을 환영합니다! 
우리는 1인 창작자와 소상공인의 마케팅 업무를 자동화하는 "무인 마케팅 에이전트" 생태계를 만들어가고 있습니다. 여러분의 모든 기여가 큰 힘이 됩니다.

## 🤝 메인테이너의 역할

본 프로젝트의 메인테이너는 생태계의 보안과 코드 품질을 유지하는 중요한 책임을 맡고 있습니다.
* 풀 리퀘스트(PR) 검토 및 병합
* 이슈(Issue) 분류 및 답변
* 릴리스 관리 및 버전 배포
* 프롬프트 엔지니어링 템플릿 검증

## 🛠️ 기여할 수 있는 분야

개발자뿐만 아니라 마케터, 기획자 등 누구나 기여할 수 있습니다.
1. **프롬프트 템플릿 제안**: 각 LLM(ChatGPT, Claude, Gemini)에 최적화된 마케팅 자동화 프롬프트.
2. **코드 개선**: 다중 LLM 라우팅 로직 개선, UI/UX 버그 수정.
3. **문서 번역 및 보완**: 다국어 지원, 튜토리얼 작성.
4. **보안 취약점 점검**: (발견 시 `SECURITY.md`에 명시된 연락처로 개별 보고)

## 📌 기여 워크플로우

1. 이 저장소를 **Fork** 합니다.
2. 기능 추가 또는 버그 수정을 위한 **Branch**를 생성합니다. (`git checkout -b feature/amazing-feature`)
3. 변경 사항을 커밋합니다. (`git commit -m 'Add some amazing feature'`)
4. 브라우치에 푸시합니다. (`git push origin feature/amazing-feature`)
5. **Pull Request**를 엽니다.

## 📝 Pull Request 규칙

* PR 제목은 변경 사항을 명확히 요약해야 합니다.
* 본문에 어떤 문제를 해결하는지, 테스트는 어떻게 진행했는지 작성해 주세요.
* 리뷰어가 쉽게 검토할 수 있도록 커밋 내역을 논리적으로 분리해 주세요.

여러분의 참여로 더욱 강력한 무인 마케팅 에이전트 생태계가 만들어집니다. 감사합니다!
